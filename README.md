# Investment Dashboard Demo

**Description:**  
A modern Investment Dashboard demo built with HTML, CSS, and PHP. This project simulates deposits, withdrawals, transfers, live profit calculation, and transaction history. Demo mode allows testing without affecting real data—no real money is involved.

---

## Features

- **User Authentication:** Login system with session handling  
- **Demo Mode:** Test features without affecting real data  
- **Dashboard:** View balances, live profits, and user statistics  
- **Transactions:** Track deposits, investments, profits, and withdrawals  
- **Transfers:** Simulated user-to-user transfers with PIN verification  
- **Withdrawal Requests:** Request withdrawals with PIN validation  
- **Responsive Design:** Works seamlessly on mobile, tablet, and desktop  

---

## Tech Stack

- **Frontend:** HTML5, CSS3  
- **Backend:** PHP  
- **Database:** MySQL (for live mode; demo mode uses simulated data)  
- **Hosting/Deployment:** GitHub + Render
